# qa-automation-final-project
Итоговый проект по курсу "Автоматизация тестирования с помощью Selenium и Python" (https://stepik.org/course/575/syllabus)
Тесты для ревью промаркированы @pytest.mark.need_review и могут быть запущены по команде -m need_review
Тесты подготовленные для ревью:
test_user_can_add_product_to_basket
test_guest_can_add_product_to_basket
test_guest_cant_see_product_in_basket_opened_from_product_page
test_guest_can_go_to_login_page_from_product_page

Оставленны только те комментарии, которые считались мною необходимыми или важными т.к. это учебный проект.
